# spectator
Interface to Spectator Earth API
# Description
This package provides interface to Spectator Earth API, mainly for obtaining the acquisition plans and satellite overpasses 
for Sentinel-2, Landsat-8 and Sentinel-1 satellites. Current position and trajectory can also be obtained for a much larger set of satellites.
